# Geospatial-and-Machine-Learning-Based-Irrigation-Schedule-Modelling-for-Rabi-Maize-in-North-Bihar
Geospatial &amp; ML-based irrigation scheduling for Rabi maize in North Bihar using Sentinel-1/2, GEE, and Random Forest/XGBoost/LightGBM for crop classification, CWR/IWR estimation, and irrigation planning.
